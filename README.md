# ☕ Projeto Cafeteria

Uma landing page para uma cafeteria fictícia, desenvolvida com foco em design visual e experiência do usuário. O projeto apresenta vídeo de fundo, seção "Sobre Nós", mapa de localização e navegação com modal interativo.

---

## 🚀 Tecnologias Utilizadas

- **HTML5** — Estrutura semântica da página
- **CSS3** — Estilização e layout
- **JavaScript** — Interatividade (modal de endereço)
- **Google Fonts** — Tipografia (Roboto)
- **Google Maps Embed** — Mapa de localização

## ✨ Funcionalidades

- Vídeo de fundo com máscara de sobreposição
- Header com navegação e logo
- Seção hero com chamada para ação
- Seção "Sobre Nós" com imagem e texto
- Modal com mapa do Google Maps para exibir o endereço
- Fonte personalizada via Google Fonts

## 💻 Como Rodar Localmente

1. Clone o repositório:
```bash
   git clone https://github.com/Almirodesouza/Projeto-Cafeteria.git
```

2. Acesse a pasta do projeto:
```bash
   cd Projeto-Cafeteria
```

3. Abra o arquivo `index.html` no navegador.

> Não é necessário instalar dependências — o projeto é 100% front-end estático.
